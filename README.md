# Atividade 2 de Web - Manipulação de DOM (Conceitos Básicos)

Este repositório contém uma atividade prática focada no estudo do **DOM (Document Object Model)** utilizando JavaScript Vanilla. O objetivo é demonstrar a interação entre scripts e elementos HTML de forma dinâmica.

## 🚀 Funcionalidades

O projeto é dividido em três módulos de interação:

1.  **Contador de Cliques:**
    * Botões de incremento e decremento.
    * Validação lógica para impedir números negativos.
2.  **Texto Dinâmico e Contador de Caracteres:**
    * Monitoramento de input em tempo real.
    * Contagem de caracteres ignorando espaços vazios (utilizando Regex).
    * Criação de parágrafos dinâmicos ao pressionar a tecla `Enter`.
3.  **Gerador de Listas Dinâmicas:**
    * Seleção entre listas ordenadas (`<ol>`) e não ordenadas (`<ul>`).
    * Inserção de itens de lista (`<li>`) via manipulação de elementos pai.
4.  **Reset de Sistema:**
    * Botão para limpar todos os estados, variáveis e elementos gerados dinamicamente.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura da página e elementos de formulário.
* **CSS3:** Estilização básica e layout.
* **JavaScript (ES6+):** Lógica de manipulação, escuta de eventos e controle de estado.

## 💡 Conceitos de JavaScript Aplicados

Neste projeto, foram exercitados os seguintes conceitos:

* **Seleção de Elementos:** Uso de `document.getElementById()`.
* **Manipulação de Eventos:** `addEventListener` para eventos de `click`, `input` e `keydown`.
* **Criação de Elementos:** Uso de `document.createElement()` e `appendChild()`.
* **Lógica de Strings:** Manipulação com `.replace(/\s/g, '')` e `.trim()`.
* **Manipulação de Conteúdo:** Uso de `.innerText` e `.innerHTML`.
